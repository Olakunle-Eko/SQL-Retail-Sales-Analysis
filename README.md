# SQL-Retail-Sales-Analysis
The project involves setting up a retail sales database,  performing exploratory data analysis (EDA), and answering specific  business questions through SQL queries.
# Questions answered
1. What country generated the most revenue?

2. What country generated the lowest revenue?

3. What months had the most revenue? top 3

4. Which customers generated the most revenue? top 10

5. Which 10 countries had the highest demands for products and revenue generated?

6. What percentage of customers are repeat customers? determine those ordering the same products and those ordering different products

7. Determine total number of customers
