# SQL-Retail-Sales-Analysis
The project involves setting up a retail sales database,  performing exploratory data analysis (EDA), and answering specific  business questions through SQL queries.
# Questions answered
what country generated the most revenue?

what country generated the lowest revenue?

what months had the most revenue? top 3

which customers generated the most revenue? top 10

which 10 countries had the highest demands for products and revenue generated?

what percentage of customers are repeat customers? determine those ordering the same products and those ordering different products

determine total number of customers
